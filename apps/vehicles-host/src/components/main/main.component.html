Using the router
Loading module federated components via the router is straight forward. Basically, just like lazy loading from a remote entry.

Loading webcomponents
Webcomponents can be added, too. They have advantages and disadvantages. They are isolated in themselves, which makes them ideal when you want to avoid a lot of team sync. But they can become more heavy over time, as they do not share anything with the host. Also, nx support is not as great.
